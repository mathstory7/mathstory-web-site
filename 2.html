✅ What is CSS? CSS stands for Cascading Style Sheets. It’s the language used to describe how HTML elements should look on a web page. If HTML is the structure (the skeleton) of a website, then CSS is the style (the clothes and design). ✅ What CSS Does Colors & Backgrounds: text color, background color, gradients Fonts & Text: font family, size, bold, italic, spacing Layout: position of elements, margins, padding, alignment Responsive Design: how pages look on phone vs. desktop Animations: transitions, hover effects, keyframe animations
